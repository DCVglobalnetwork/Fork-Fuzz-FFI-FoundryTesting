## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Cast

```shell
$ cast 
```

### Help

```shell
$ forge --help
$ anvil --help
$ cast --help
```
